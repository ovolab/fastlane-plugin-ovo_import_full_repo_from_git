# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-08

### Added
- Initial release of `ovo_import_full_repo_from_git`
- Support for cloning a full remote git repository before importing a Fastfile
- Support for importing a Fastfile from a custom path inside the checked out repository
- Support for branch-based checkout
- Support for SemVer/tag-based version selection
- Support for optional clone caching via `cache_path`
- Support for optional submodule initialization
- README with installation and usage examples
